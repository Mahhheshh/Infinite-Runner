## installation:
- clone repo locally using git clone `https://github.com/Mahhheshh/Run4Fun.git`
- install python requirements by doing `pip install -r requirement.txt`
- then run `python src\main.py`

## Controls
| Keys  | Description           |
|----   | ----                  |
| S     | Slide                 |
| SPACE | Jump                  |
| P     | Pause                 |
| ESP   | Open help/Menu screen |
